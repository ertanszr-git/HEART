# HTTP Error Alert and Recording Tool - HEART

HEART is a script for monitoring port 80 (HTTP port) and recording all HTTP errors.

## Installation

You have to use Debian based distributions for this script.

```bash
./install.sh
```

## Usage

```bash
./alert.sh
```
